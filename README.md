# LittleFlightAnnouncer
Freeware add-on utility for MSFS 2024
Author: Swede001

## Installation Instructions

1. **ZIP file**\
   Unzip the zip archive. There are two folders in the archive:
   -  LittleFlightAnnouncer
   -  routeadapter-wasm-misc

1. **LittleFlightAnnouncer**\
   This folder can be placed anywhere. The EXE file is the main application.

3. **routeadapter-wasm-misc**
   - Copy the entire folder into your MSFS Community folder.
     - Typical location: `C:\Users\<YourUsername>\AppData\Local\Packages\Microsoft.FlightSimulator_...\LocalCache\Packages\Community`
   - Restart MSFS if itâ€™s already running.

This freeware utility helps MSFS 2024 users by adding crew announcements to airline flighs.
